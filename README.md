# ArogyaConnect - Medicare Hackathon Project

A complete, beautiful MERN stack application built for rural healthcare.

## Features Let's Build

- **AI Doctor (Primary Focus)**: Interactive 3D visual human model with context-aware chat, medical scan UI, and WhatsApp style UI. Built using ThreeJS and React Three Fiber.
- **Consults**: Connect with experts over video.
- **Vitals**: Smart SpO2 and Heart Rate tracking using mobile flashlight.
- **Events & Emergency**: One-tap ambulance routing and community health camps.

## Project Structure

- `/client` - Vite + React + Tailwind + Three.js
- `/server` - Express + Mongoose

## How to Run

### 1. Client (Frontend)
```bash
cd client
npm install
npm run dev
```

### 2. Server (Backend)
```bash
cd server
npm install
npm run start
```
